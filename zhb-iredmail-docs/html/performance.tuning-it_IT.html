Configura un server DNS in rete o localmente per mettere in cache

le richieste

I server di posta usano pesantetemente il server DNS ed effettuta molte, molte richieste al DNS che esso sia inrete locale o sullo stesso server può aiutare molto:

  • Velocizza le richiste DNS. Questo aiuta molto.
  • riduce le richieste DNS verso i server DNSBL, così che possiate usare i loro eccellenti servizi senza superare il limite massimo di interrogazioni.

Attiva il servizio postscreen aiuta a ridurre lo spam

  • Attiva servizio postscreen

Se non vuoi usare il servizio postscreeen puoi, alternativamente, abilitare il servizio DNSBL, anche questo può aiutare parecchio. Anche se entrambi i servizi postscreen e DNSBL puro utilizzano gli stessi server DNSBL, postscreen offre soluzioni aggiuntive per ridurre lo modalità postscreen è migliore.

postscreen ed il servizio DNSBL aiutano ad intercettare molto spam prima ancora che lo spam venga inviato alla coda della posta, risparmiando cosi un bel po' di risorse del sistma.
